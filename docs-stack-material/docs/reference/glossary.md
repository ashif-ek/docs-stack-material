# Glossary

Terms used across the project documentation.

*   **JWT (JSON Web Token)**: Compact URL-safe means of representing claims to be transferred between two parties.
*   **MVT (Model-View-Template)**: Architectural pattern used by Django.
*   **SPA (Single Page Application)**: Web app that interacts with the web browser by dynamically rewriting the current web page with new data from the web server.
*   **ORM (Object-Relational Mapping)**: Technique for converting data between incompatible type systems using object-oriented programming languages.
*   **Hook**: Special function in React that lets you “hook into” React features.
