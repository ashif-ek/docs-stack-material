# Configuration Options

Detailed reference for environment variables.

| Variable | Description | Required | Default |
| :--- | :--- | :--- | :--- |
| `DEBUG` | Enable debug mode | No | `False` |
| `SECRET_KEY` | App secret for signing | Yes | - |
| `DATABASE_URL` | DB Connection string | Yes | - |
| `ALLOWED_HOSTS` | Comma-separated domains | Yes | `*` |
| `CORS_ORIGIN` | Frontend URL for CORS | Yes | - |
| `PORT` | Server port | No | `8000` |

See [Configuration](../configuration.md) for setup instructions.
