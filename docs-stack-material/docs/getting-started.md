# Getting Started

Welcome to the documentation for Ashif's Project Portfolio. This collection showcases a diverse range of open-source projects, from full-stack web applications to utility tools and data analysis platforms.

## What's Included?

This site mainly documents the following key projects:

*   **[Portfolio React](projects/github_repos/portfolio-react.md)**: A personal developer portfolio with an admin dashboard.
*   **[Noirel Ecommerce](projects/github_repos/noirel-ecommerce.md)**: A high-end e-commerce experience.
*   **[Salary Checker](projects/github_repos/salary-checker.md)**: AI-driven salary insights and predictions.
*   **[Time Lens](projects/github_repos/time-lens-python.md)**: A psychological tool for time perception.

Each project documentation includes its specific features, tech stack, and installation guides.

## Prerequisites

Most projects in this portfolio rely on the following technologies. Ensure you have them installed before running them locally:

*   **Node.js** (v18+ recommended) for React/Next.js projects.
*   **Python** (v3.10+) for Django/FastAPI backends.
*   **Docker** for containerized applications.
*   **Git** for version control.
